Crafter Social UI
=================

Crafter Social Front End.

-------------

2. Execute: `npm install -g yo`

Other Tasks
-----------

Run the app: `$ grunt server`

Build the app: `$ grunt`

Libraries
---------

Crafter Social UI uses the following libraries

* jQuery
* Handlebars
* Backbone
* underscore
* CKEditor
* Twitter Bootstrap (for sass)
* modernizr
* requirejs
